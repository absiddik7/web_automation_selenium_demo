# Selenium Web Automation Project with Reporting

This project is a Selenium-based automation framework using Python, `pytest`, and Allure for test reporting with video and screenshot capture on failures.

## Prerequisites

- Python 3.x

## Getting Started

1. **Clone the Repository:**

    ```bash
    git clone https://github.com/absiddik7/web_automation_selenium_demo.git
    cd web_automation_selenium_demo
    ```

2. **Set Up Virtual Environment:**

    **For Windows:**
    ```bash
    python -m venv .venv
    .venv\Scripts\activate
    ```

    **For macOS/Linux:**
    ```bash
    python3 -m venv .venv
    source .venv/bin/activate
    ```

3. **Install Requirements:**

    ```bash
    pip install -r requirements.txt
    ```

4. **Run the Project:**

    ```bash
    python main.py
    ```
